# いるまオープン議会について：議会のリアルを、市民の手に。
「いるまオープン議会」は、入間市議会の膨大な議論をAI（人工知能）とオープンデータの力で整理し、誰もが街の未来を判断できる「武器」を提供するための有志市民プロジェクトです。
## 1. 立ち上げの経緯：活動の変遷と「3つの壁」
本プロジェクトは、長年にわたる情報発信の試行錯誤から誕生しました。これまでX（旧Twitter）での「フロー型」発信や、Word・Googleサイト・Notionを用いた「ストック型」の公開を行ってきましたが、市民へ正確な情報を届ける上で、常に以下の3つの壁が立ちはだかっていました。
 * **【作業量の壁】膨大なリサーチコスト**
   年間数千ページに及ぶ議事録を読み込み、要点を抽出して構造化する作業は、人力では限界に近い時間を要していました。
 * **【文脈の壁】断片化する議論の歴史**
   議会は数年単位で一つの課題を議論します。点在する過去の議事録を遡り、いつ、誰が、何を言い、どう方針が転換されたのかという「ストーリー」として提示することは極めて困難でした。
 * **【証拠の壁】一次情報へのアクセスの欠如**
   従来のまとめ記事は、作成者の「解釈」が先行しがちであり、読んだ市民が「本当にそう言っているのか？」と疑問を持った際、即座に生データ（証拠）を確認する術がありませんでした。
## 2. プロジェクトの本質：情報の「翻訳」と「直結」
これらの課題を解決するため、最新の生成AIを活用した独自のデータ構築プロセスを採用しています。
### AIによる「文脈」の可視化
膨大な議事録の読み込みと整理をAIがサポートすることで、数年間にわたる議論のバトンを1本の線で繋ぎ合わせます。これにより、行政の「朝令暮改」や、議員の一貫した「追及」の足跡を浮き彫りにします。
### 「要約」と「証拠」のシームレスな統合
本サイトでは「3分で読める分かりやすい要約」と「実際の議事録からの生引用（証拠）」をワンクリックで行き来できる構造にしています。誰かの解釈だけで終わらせず、常に市民が「一次情報」に直接アクセスできる透明性を保証します。
## 3. 構成の型：3段構成のアーカイブ
すべての地域課題（イシュー）ページは、以下のMECE（モレなく、ダブリなく）な3段構成で作成されています。

| 構成 | 役割 | 対象 |
| :--- | :--- | :--- |
| **第1段：要約** | 3分で現状と論点を把握する | 全市民・多忙な方 |
| **第2段：時系列** | 議論の変遷とストーリーを追う | 経緯を知りたい方 |
| **第3段：証拠** | 議事録の生データで真偽を確かめる | 監視・分析したい方 |

## 4. 今後の展望とロードマップ
私たちは、このサイトを市民と共に成長する「動的なデータベース」へと進化させます。
 1. **双方向のファクトチェック**:
   市民からの感想や専門的なファクトチェックをフォーム経由で受け付け、それらをObsidianへ自動格納。フィードバックに基づき、AIと人間が協力して記事をリアルタイムでアップデートします。
 2. **議員・会派名鑑の拡充**:
   「誰が」「いつ」「何を」発言したかを抽出した発言ログを強化し、選挙や市民活動の判断材料となるデータベースを構築します。
 3. **地域課題の網羅**:
   新庁舎、学童、財政といった重要テーマから、身近な地域課題まで、入間市のすべての議論をインデックス化します。
