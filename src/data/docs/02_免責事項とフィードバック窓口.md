# 免責事項とフィードバック窓口

「市議会議員通信簿」をご利用いただきありがとうございます。本サイトは市民の皆様の市政への関心を高めることを目的としたオープンデータ活用プロジェクトです。
情報の透明性と正確性には最大限努めておりますが、ご利用にあたっては以下の免責事項をご確認ください。

## 1. 免責事項

1. **AIによる機械的抽出の限界**
   本通信簿における「公約進捗率」や「発言テーマの要約・分類」は、公開されている入間市議会の議事録原本（テキストデータ）および選挙公報を基に、AI（人工知能）が機械的に抽出・集計したものです。
   AIの特性上、発言の微妙なニュアンスや文脈が完全に反映されていない場合や、解釈の揺らぎによる誤分類が含まれる可能性があります。

2. **内容の完全性の非保証**
   本サイトの情報は、議会活動の「すべて」を網羅するものではありません。委員会での発言、水面下での調整、市民からの直接相談への対応など、一般質問の議事録に表れない議員の重要な活動は評価の対象外となっています。

3. **利用による損害について**
   本サイトの情報を利用したことによって生じたいかなる損害についても、作成者および運営チームは一切の責任を負いません。投票行動等の参考とされる場合は、必ず入間市議会の公式情報や議会中継、各議員の発行する公報等を併せてご確認ください。

## 2. フィードバックと訂正申請について

本通信簿は「完成品」ではなく、市民や議員の皆様との対話を通じて改善していく「β版」のプロジェクトです。
もし、以下のようなケースにお気づきの場合は、ぜひフィードバック窓口よりお知らせください。

* **事実誤認の指摘**: 「この一般質問は、公約の〇〇項目に該当するはずだがチェックが漏れている」
* **要約の修正依頼**: 「質問の要旨が本来の意図と異なる要約になっている」
* **機能へのご要望**: 「もっとこういうデータが見たい」「この表現は分かりにくい」
